Veri İçe Aktar

Instagram Verilerini İndir

  1. Instagram'da Ayarlar → Hesap Merkezi'ne git
  2. Bilgileriniz ve izinleriniz → Bilgilerinizi indirin
  3. Bazı bilgilerinizi indirin seçeneğini seç
  4. Kaydedilen gönderiler veya Saved kutusunu işaretle
  5. Format: JSON seç ve indir
  6. ZIP dosyasından saved_posts.json dosyasını çıkar

Instagram JSON dosyasını yükle

saved_posts.json veya saved_saved_media.json

X (Twitter) Verilerini İndir

  1. X.com'da Ayarlar → Hesabınız'a git
  2. Verilerinizin bir arşivini indirin seçeneğine tıkla
  3. Şifrenizi girin ve arşivi talep edin
  4. E-posta ile indirme linki gelecek (1-2 gün sürebilir)
  5. ZIP'ten data/bookmarks.js dosyasını çıkar
  6. Dosyayı metin editöründe aç, başındaki window.YTD.bookmark.part0 = kısmını sil

X/Twitter JSON dosyasını yükle

bookmarks.json veya düzenlenmiş bookmarks.js

JSON dosyasını sürükleyin veya tıklayın

Desteklenen format: JSON

Örnek JSON Formatı:

{
  "bookmarks": [
    {
      "url": "https://instagram.com/p/xxx",
      "title": "Başlık",
      "platform": "instagram",
      "original_author": "@kullanici"
    }
  ]
}
